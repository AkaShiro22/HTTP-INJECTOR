# HTTP-INJECTOR

HTTP Injector ini khusus untuk para pecinta Internet Gratis sekaligus Para Upreker,
Script ini menggunkan Manual Payload.
## Support Inject
1. Mode Proxy
2. Direct Proxy
3. ICMP with Proxy
4. ICMP Direct
5. SSL Direct
6. SSL Proxy

## Support Payload Command
 [crlf*2] , 
 [crlf], 
 [lfcr], 
 [cr], 
 [lf], 
 [host], 
 [port], 
 [method], 
 [host_port], 
 [protocol], 
 [netData], 
 [raw], 
 [real_raw], 
 [ssh], 
 [ua]
